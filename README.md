# 3JS-Gradient

A mesmerizing 3D animation created with Three.js, featuring rotating particles, geometric spheres, and dynamic gradient lighting effects.

## Description

This project showcases a Three.js scene with:
- Thousands of randomly positioned and rotating tetrahedron particles
- A large solid icosahedron sphere
- A wireframe icosahedron sphere
- Gradient lighting using multiple directional lights in cyan and purple hues

The animation creates a hypnotic visual experience with smooth rotations and color gradients.

## Features

- Particle system with 1000 tetrahedron meshes
- Two icosahedron geometries (solid and wireframe)
- Ambient and directional lighting setup
- Responsive design that adapts to window resize
- High-performance WebGL rendering
